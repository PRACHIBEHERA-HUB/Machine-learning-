#  House Price Prediction using Machine Learning

##  Project Overview
This project predicts house prices using Machine Learning based on various property features such as area, number of bedrooms, bathrooms, stories, parking, and other housing attributes. The objective is to build a predictive model that estimates house prices accurately using historical housing data.

##  Objective
- Predict house prices using Machine Learning.
- Perform data preprocessing and feature engineering.
- Train and evaluate a regression model.
- Visualize the relationship between different features and house prices.

## Dataset
The dataset contains information about houses, including:
- Area
- Bedrooms
- Bathrooms
- Stories
- Parking
- Main Road
- Guest Room
- Basement
- Air Conditioning
- Preferred Area
- Furnishing Status
- Price (Target Variable)

##  Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

##  Project Workflow
1. Import required libraries.
2. Load the housing dataset.
3. Perform data cleaning and preprocessing.
4. Encode categorical variables.
5. Split the dataset into training and testing sets.
6. Train the Machine Learning model.
7. Predict house prices.
8. Evaluate model performance using regression metrics.
9. Visualize the results.

##  Model Used
- Linear Regression

##  Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

##  Features
- Data Cleaning
- Data Preprocessing
- Feature Encoding
- Model Training
- Price Prediction
- Model Evaluation
- Data Visualization




#  Salary Prediction using Machine Learning

##  Project Overview
This project predicts employee salaries based on years of experience using Machine Learning. The goal is to build a regression model that estimates salary accurately and demonstrates the relationship between experience and income.

##  Objective
- Predict salary based on years of experience.
- Understand the relationship between experience and salary.
- Train and evaluate a Machine Learning regression model.
- Visualize the prediction results.

##  Dataset
The dataset contains:
- Years of Experience
- Salary (Target Variable)

##  Technologies Used
- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

##  Project Workflow
1. Import required libraries.
2. Load the salary dataset.
3. Explore and preprocess the data.
4. Split the dataset into training and testing sets.
5. Train the Linear Regression model.
6. Predict employee salaries.
7. Evaluate model performance.
8. Visualize actual vs predicted salaries.

##  Machine Learning Model
- Linear Regression

##  Evaluation Metrics
- Mean Absolute Error (MAE)
- Mean Squared Error (MSE)
- R² Score

##  Features
- Data Preprocessing
- Model Training
- Salary Prediction
- Model Evaluation
- Data Visualization
- Regression Analysis
